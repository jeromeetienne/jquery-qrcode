jquery.qrcode.js

jquery.qrcode.js é plugin jquery para a geração de qrcode puramente no browser. Ele permite a você facilmente adicionar qrcode a sua página web. Ele roda independente, fica menor que 4k após minify+gzip, sem imagens para download. Ele não depende de serviços externos que ficam indo e vindo, ou adiciona uma demora ao carregamento. É baseado em uma biblioteca que cria qrcode em várias linguagens. jquery.qrcode.js inclui isto para facilitar a inclusão no seu próprio código.

Mostre, não fale, aqui está um exemplo

Como usar

Deixe-me mostrar o caminho para você. Primeiramente inclua ele em sua página com a tag padrão

<script type="text/javascript" src="jquery.qrcode.min.js"></script>

Então crie um elemento DOM que irá conter a imagem do qrcode gerada. Vamos usar uma div

<div id="qrcode"></div>

Então você adiciona o qrcode neste container

jquery('#qrcode').qrcode("este plugin é demais");

É isto. veja em ação.

Conclusão

jquery.qrcode.js está disponível no github aqui sob uma licença MIT. Se achar algum bug, modifique-as no github. Sinta-se livre para forkar, modificar e se divertir com isto :)
